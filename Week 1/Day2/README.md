# Day 2 - SQL Practice

## 📌 Overview
On Day 2, I practiced SQL queries using an `Employees` table.  
The focus was on learning how to retrieve, filter, group, and analyze data using different SQL clauses and operators.

---

## 📚 Topics Covered

### 🔹 SELECT Statements
- Fetching all records
- Selecting specific columns
- Viewing employee details

### 🔹 WHERE Clause
- Filtering records using conditions
- Salary-based filtering
- Department and city filtering
- Experience-based conditions

### 🔹 GROUP BY
- Grouping data department-wise
- Using aggregate functions:
  - SUM()
  - AVG()
  - MAX()
  - MIN()
  - COUNT()

### 🔹 HAVING Clause
- Filtering grouped data
- Applying conditions on aggregate values

### 🔹 TOP Queries
- Retrieving highest salary employees
- Finding most experienced employees

### 🔹 DISTINCT Keyword
- Finding unique values
- Removing duplicate records

### 🔹 Comparison Operators
- `>`
- `<`
- `>=`
- `<=`
- `<>`

### 🔹 Logical Operators
- `AND`
- `OR`
- `NOT`

### 🔹 IN and NOT IN
- Filtering multiple values in a single query

### 🔹 BETWEEN Operator
- Selecting values within a range

### 🔹 LIKE Operator
- Pattern matching using:
  - `%`
  - `_`


## 🚀 Learning Outcome
By completing Day 2 practice, I improved my understanding of:
- Writing SQL queries
- Filtering and grouping records
- Using SQL operators effectively
- Analyzing tabular data using SQL
